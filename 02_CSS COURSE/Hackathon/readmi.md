# STRIKE - Premium Course Platform

## Project Overview
STRIKE is an interactive website for an upcoming premium paid course platform featuring Data Structures & Algorithms (DSA), Generative AI, and interview-focused programs. The website showcases a unique horror-themed design with interactive elements that transform based on user choices.

## Features

### Interactive Theme System
- **Default Mode**: Standard educational platform interface
- **Horror Mode**: Activated when "Not Interested" is selected
  - Spooky red color scheme with flickering effects
  - Animated horror eyes that follow cursor
  - Creepy font styling and animations
  - Dark forest background
- **Heaven Mode**: Activated when "Join Now" is selected
  - Serene blue color scheme
  - Elegant cursive fonts
  - Angelic background imagery
  - Smooth transitions and animations

### Navigation
- Fixed navigation bar with blur effect
- Responsive menu for mobile, tablet, and desktop
- Logo with animated text effects
- Smooth scrolling to sections

### Hero Section
- Animated hand and STRIKE logo
- Interactive call-to-action buttons
- Theme-responsive content display
- Horror eyes animation in horror mode

### About Section
- Course description and details
- Instructor profile (Rohit Negi)
- Animated profile image with hover effects
- Educational background and achievements

### Courses Section
- Interactive 3D flip cards for course display
- Two main courses:
  - DSA Course in C++
  - GEN AI Course
- Hover animations and detailed backface information
- Theme-appropriate styling

### Footer
- Company information and social links
- Quick navigation links
- Legal information
- Responsive grid layout

## Theme System Details

### Horror Mode Activation
- Triggered by selecting "Not Interested"
- Changes fonts to creepy styles
- Adds red color scheme with flickering effects

- Modifies button behaviors and appearances

### Heaven Mode Activation
- Triggered by selecting "Join Now"
- Implements elegant cursive fonts
- Blue color scheme throughout
- Changes background to heavenly imagery
